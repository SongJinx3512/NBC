# txtRPG
콘솔 텍스트 rpg

---

## 프로젝트 소개
VS 2022의 콘솔창에서 간단히 RPG 게임의 요소의 일부를 구현 해본 게임

---

## 개발기간
2023/08/24

---

팀원 : 송유섭 - txtRPG

---

Visual C++ 2022 - 17.7.2
C# 도구 - 4.7.0
Visual Studio Tools for Unity - 17.7.0.0
Framework : .NET Framework - 4.8.09032

---

## 주요 기능
### 게임 시작 화면
간단한 게임 소개

마을에서 할수있는 행동 표시

### 상태보기
* 캐릭터 정보 표시

* 행동 표시

### 인벤토리

* 보유 중인 아이템 표시

* 장착한 아이템은 [E]로 표시

### 장착관리
* 장착 해제 기능

* 장착 표시 [E]
